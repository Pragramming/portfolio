Jayesh Kaushalya Portfolio

What this is
- A static portfolio site customized from the resume details in `../resume.txt`.
- A professional portfolio with a contact form powered by Web3Forms.

How to run
- Open `index.html` in a browser.

Included sections
- Hero summary
- Selected GenAI and decision science work
- Experience
- Education and technical skills
- Contact form

Notes
- Replace `REPLACE_WITH_YOUR_WEB3FORMS_ACCESS_KEY` in `index.html` with your real Web3Forms access key.
- The form sends directly from the site to Web3Forms, which then forwards the submission to your email.
- No SMTP or custom backend is required for this setup.
