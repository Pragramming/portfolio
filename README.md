Excalidraw-like static scaffold (no Node required)

What this is
- A minimal web-based drawing app using Fabric.js loaded from CDN.
- Features: draw, select, rectangle, circle, line, delete, undo, clear, export PNG, save/load JSON.

How to use locally
- Open `index.html` in a browser (double-click or drag into browser). No build step or Node required.

Deploy to Render (static site)
1. Push this folder to a GitHub repository.
2. On Render, create a new "Static Site" and connect the repo.
3. Set the build command to empty and the publish directory to `/` (root).
